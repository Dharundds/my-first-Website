PyInstaller

This document is a beginner guide about PyInstaller

What is Pyinstaller?

To put it in simple terms, Pyinstaller is a module in Python which bundles a Python application or script which you have wrote and all its dependencies into single package.

Converts .py to .exe

Why PyInstaller? What does this module do?

This module makes the user to run their packaged app without installing any of such Python interpreter or any modules. It correctly bundles the major Python packages such as Django, wxPython, numpy and so on.

Compatible python versions

Versions supported for Pyinstaller is Python 3.5 or above.

How to Install Pyinstaller using pip

Within your command prompt or within your environment terminal type the command.

>>>pip/pip3 install pyinstaller

To check if pyinstaller is installed properly

>>> pyinstaller –v

If installed properly it will give the version of pyinstaller. If not make sure that execution path includes the proper directory:

Window: C:\PythonXY\script_name
XY stands for versions (EX: Python34 for Python 3.4)
Linux: /usr/bin
OS X: /usr/bin OR /usr/local/bin

How to use pyinstaller using command prompt

>>>pyinstaller script_name.py

We can add additional parameters

-w

The python console window does not appear

>>>pyinstaller –w script_name.py

--onefile

Makes sure that all the modules in our python program are included in one executable file than multiple files.

>>>pyinstaller --onefile script_name.py

--add-data

It includes all the dependencies like databases, images, other python files etc.

>>>pyinstaller --add-data “database.db” script_name.py

--icon

It will add icon to the executable

>>>pyinstaller --icon=name.ico script_name.py

We can include the above parameters in one single line

>>>pyinstaller –w --add-data --onefile script_name.py
